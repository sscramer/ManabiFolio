# AI指導要録生成機能のセットアップガイド

この機能は、Google Gemini APIを使用して、生徒の振り返り回答データから「生徒指導要録」の文章案（総合所見・特別活動の記録）を自動生成するものです。

## 1. Gemini APIキーの取得

1. [Google AI Studio](https://aistudio.google.com/) にアクセスします。
2. "Get API key" をクリックし、新しいAPIキーを作成します。
3. キーをコピーしておきます。

## 2. Google Apps Scriptへの設定

セキュリティのため、APIキーはコードに直接記述せず、「スクリプトプロパティ」に保存します。

1. Google Apps Scriptエディタを開きます（スプレッドシートの「拡張機能」>「Apps Script」など）。
2. 左側のメニューから「プロジェクトの設定」（歯車アイコン `⚙️`）をクリックします。
3. 画面下部の「スクリプト プロパティ」セクションまでスクロールします。
4. 「スクリプト プロパティを追加」をクリックします。
5. 以下の通り入力して保存します。
    - **プロパティ**: `GEMINI_API_KEY`
    - **値**: (取得したAPIキー)
6. 必要に応じて、学校種別や文体の前提を `AI_SCHOOL_CONTEXT` に設定します。
    - **例**: `日本の学校の教員`

## 3. 使用方法

1. Webアプリ（教員用ダッシュボード）を開きます。
2. 画面上部の**「AI指導要録作成」**ボタンをクリックします（紫色のボタン）。
3. ドロップダウンリストから対象の生徒（Email）を選択します。
4. **「✨ AI生成を実行する」**ボタンをクリックします。
5. 数十秒間、AIが全回答データを分析し、文章案を生成します。
6. 生成されたMarkdownテキストが表示されるので、コピーして使用してください。

## 注意事項

- 生成される文章はあくまで「案」です。必ず教員が内容を確認・修正した上で使用してください。
- 生徒の回答データがプロンプトとして外部API（Google Gemini）には送信されますが、モデルの学習には使用されません（エンタープライズ/有料プラン等の場合。無料枠の使用時はGoogleの規約を確認してください）。
- 組織のプライバシーポリシーで外部AI利用が許可されている場合のみ有効化してください。
- 回答データが多い場合、処理に時間がかかることがあります。
