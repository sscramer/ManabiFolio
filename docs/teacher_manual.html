1. はじめに

教員アカウントでログインすると、セッション管理や設問の作成など、教員専用の機能が利用できます。

教員メイン画面

教員としてログインすると、上部にナビゲーションタブが表示されます：

  • 振り返りポートフォリオ：生徒に表示される画面を確認（プレビュー用）
  • 読書履歴調査：生徒への見え方を確認（プレビュー用）
  • 教員ダッシュボード：各種管理機能へアクセス

2. 教員ダッシュボード

教員ダッシュボードでは、以下の機能にアクセスできます：

  • 一覧・管理：セッションや設問の管理
  • データ・分析：読書履歴の集計・分析
  • 設定・出力：生徒名簿の管理やデータ出力

3. 振り返りポートフォリオ（プレビュー）

概要

設問が生徒にどのように表示されるかを確認するためのプレビュー画面です。

注意: この画面は生徒の回答を確認する機能ではありません。生徒への見え方を確認する用途です。（デバッグから成り済ました場合を除く）

操作方法

  1. 振り返りポートフォリオタブをクリック
  2. プルダウンからセッションを選択
  3. 生徒に表示される質問の見え方を確認

4. 読書履歴調査（プレビュー）

概要

読書履歴調査の画面が生徒にどのように表示されるかを確認するためのプレビュー画面です。

注意: この画面は生徒の読書記録を確認する機能ではありません。生徒への見え方を確認する用途です。（デバッグから成り済ました場合を除く）

操作方法

  1. 読書履歴調査タブをクリック
  2. 生徒に表示される画面の見え方を確認

5. セッション管理

概要

振り返りセッション（設問セット）の管理を行います。

機能

  • セッション一覧：作成済みセッションを確認
  • 新規作成：新しいセッションを作成
  • 編集：既存セッションの設問を編集
  • ステータス管理：公開/非公開の切り替え

セッションのステータス

ステータス 説明
公開中 生徒が回答可能
非公開 生徒には表示されない

6. 設問の新規作成と共通質問セット

共通質問セットとは

共通質問セットは、通年で質問したい内容についてのセットを作成する機能です。

共通セットを利用した質問作成

作成方法 説明
共通セットを利用 通年で回答できるビューでの質問が作成されます
共通セットを利用しない 単発での質問となります

操作方法

  1. 教員ダッシュボード → 一覧・管理
  2. 新規作成をクリック
  3. 共通質問セットを利用する場合は、セット選択プルダウンから選択
  4. 設問内容を入力して保存

7. 読書履歴集計

概要

生徒の読書履歴データを学期・学年・クラス別に集計できます。

操作方法

  1. 教員ダッシュボード → データ・分析 → 読書履歴集計
  2. 集計条件を選択：
    • 学期：1学期/2学期/3学期
    • 学年：対象学年
    • クラス：対象クラス（全クラスも可）
  3. 集計実行ボタンをクリック
  4. 集計結果が表示されます

8. 生徒名簿管理

概要

登録済み生徒の情報を確認・編集できます。

操作方法

  1. 教員ダッシュボード → 設定・出力
  2. 生徒名簿管理セクションで名簿を表示をクリック
  3. 生徒一覧が表示されます

表示項目

  • 生徒名
  • メールアドレス
  • 登録日
  • 編集ボタン：生徒情報の修正
